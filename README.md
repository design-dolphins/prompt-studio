# Prompt Maker

モードを選んで条件を入力するだけで、ChatGPT・画像生成AIに使えるプロンプトを自動生成するローカルWebアプリです。

## 対応モード

- 企画書・提案書
- ワイヤーフレーム（HTML / 画像出力）
- 社内制度・ルール整理
- UI/UXレビュー
- デザイン指示書
- メール・依頼文
- 議事録・要約リライト
- アイデア出し
- リサーチ・競合分析
- イラスト生成（フラットベクター）
- 自由に作る

## 起動

```bash
node server.js
```

ブラウザで開きます。

```
http://localhost:4173
```

## 特徴

- 入力内容はサーバーに送信・保存されません（ブラウザ内で完結）
- 共有リンク機能でURLに設定を保存できます
- 生成したプロンプトはそのままChatGPT等に貼り付けて使えます
